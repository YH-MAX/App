4. System Implementation

[Android App] ←→ [Spring Boot :8080] ←→ [FastAPI :8888] ←→ [MySQL]

4.1 Frontend (Android)

Technology: Java, Android SDK (API 24+), Retrofit 2, MPAndroidChart, Google Maps API

The app has 18 Activity screens organized into modules:

Module Screens Function
Auth Login, Register JWT login with email verification
Dashboard DashboardActivity Real-time sensor data, charts (5s refresh)
Alerts AlertActivity Water quality alerts (HIGH/MEDIUM/LOW)
Community Community, CreatePost, PostDetail Posts, likes, bookmarks, replies
Reports Report, SubmitReport, MapPicker Pollution reporting with location
Profile Profile, UpdateProfile, UserProfile User profiles and following

Network Layer: Uses Retrofit with JWT interceptor. API interfaces: AuthApi, WaterApi, AlertApi, CommunityApi, ReportApi, BluetoothApi.

4.2 Backend (Spring Boot)

Technology: Spring Boot 3.5.7, Spring Security + JWT, Spring Data JPA, MySQL 8.0, Java 17

Architecture: Three-tier (Controllers → Services → Repositories)

Module Function
Authentication JWT login, registration, email verification, password reset
Sensor Data Proxy to FastAPI for real-time water readings
Alerts Evaluate pH/turbidity thresholds, generate severity levels
Community Posts, likes, bookmarks, replies, follow system
Reports Pollution reports with status tracking

API Endpoints: 45+ REST endpoints across 8 groups (/api/users, /api/sensor, /api/water, /api/alerts, /api/community, /api/reports, /api/bluetooth)

Security: JWT with 24h expiration, BCrypt password hashing, email verification required

5.2 Security Testing

  • Access without token → 403 Forbidden ✓
  • Expired token → 401 Unauthorized ✓
  • Valid token → Data returned ✓
  • Passwords stored as BCrypt hashes ✓

5.3 End-to-End Testing

We tested complete user flows on a real Android phone:

  1. Registration Flow: Register → Email verification → Login → Dashboard ✓
  2. Monitoring Flow: View dashboard → Data refreshes every 5s → Alerts appear when pH < 6.5 ✓
  3. Community Flow: Create post → View in feed → Like/bookmark ✓
  4. Report Flow: Submit pollution report with location → Appears in "My Reports" ✓
